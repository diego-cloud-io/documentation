# Installing Diego 

It takes 30-50 minutes to install **Diego CD Pack**, using 5 easy commands with **Diego CLI**  🚀

With Diego CLI installed, you can proceed to execute 5 commands to perform setup for GitHub and AWS, with a range of supporting open source tools in your Kubernetes cluster. The entire process keeps GitOps at the heart of the implementation.

Please ensure prerequisites are installed for **Diego CLI**, prior to working with the CLI tool.
