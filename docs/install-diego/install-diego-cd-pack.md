# Install Diego CD Pack

It takes 30-50 minutes ⏱️ to install **Diego CD Pack**, using 5 easy commands with **Diego CLI**

As an overview, here's 5 Easy Commands to setup Diego CD Pack with Diego CLI 🚀

1  `diego init github` <br/>
2  `diego init aws` <br/>
3  `diego setup github` <br/>
4  `diego setup aws` <br/>
5  `diego install diego-core` <br/>

Please ensure to follow next steps, to deep-dive into details for these.
