diego-install-diego-core

  1. diego install diego-core

    Estimated time: 50 minutes

This updates your diego configuration file - $HOME/.diego/config with information required for further steps.

• Use with aws-context • Please note, this will yield a password for Argo CD, which can be used with the username admin.
