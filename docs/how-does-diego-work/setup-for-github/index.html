What is setup in GitHub?

GitHub setup​

• Diego installs a Github app for your organisation which sets up permissions to help implement GitOps between repositories for your applications and Diego-Core. • Diego sets up template repos in your organisation, required for Diego. These provide configuration files for applications and supporting tooling, as part of a GitOps model.
