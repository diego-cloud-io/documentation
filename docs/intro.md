# Diego Overview

Diego is a fully integrated platform which aims to streamline the microservices development process and improve cycle times. It is designed to cover every aspect of the developer process from committing the code through to deploying and monitoring the application in production. All information is centralised in one place. Diego adheres to a set of best practices for DevOps with integration of a comprehensive suite of open source tooling for security, scaling and resiliency.  

Diego is based on the GitOps principle that focuses on keeping the deployment specification and application desired state in Git. It consists of a collection of pre-configured open-source tools along with a special Diego GitOps operator that performs the orchestration and  reconciliation of application deployments from Git to Kubernetes to provide a cohesive developer experience.
